| Variable         | Cara Pengisian                                                                                                                    |
| ---------------- | --------------------------------------------------------------------------------------------------------------------------------- |
| Waktu Pengisian  | Pilih tanggal dan jam pengisian formulir                                                                                          |
| Riwayat Penyakit | Diisi riwayat penyakit pasien                                                                                                     |
| Riwayat Alergi   | Diisi riwayat alergi pasien                                                                                                       |
| Masalah          | Diisi sesuai masalah kondisi pasien                                                                                               |
| Nama Obat        | Diisi sesuai nama obat yang diberikan kepada pasien                                                                               |
| Jumlah           | Diisi sesuai jumlah obat yang diberikan kepada pasien                                                                             |
| Dosis            | Diisi sesuai dosis obat yang diberikan kepada pasien                                                                              |
| Rute             | Diisi sesuai rute obat yang diberikan kepada pasien                                                                               |
| Frekuensi        | Diisi sesuai frekuensi obat yang diberikan kepada pasien                                                                          |
| Catatan          | Diisi catatan tambahan yang diperlukan                                                                                            |
| A/T/P            | Diisi sesuai jenis pasien                                                                                                         |
| Rencana Awal     | Diisi dengan rencana tindakan farmasi yang akan dilakukan untuk memastikan terapi obat aman, efektif, dan sesuai kebutuhan pasien |

</Callout>
