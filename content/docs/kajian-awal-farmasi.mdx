<Callout title="Penting" type="warn">  
Pastikan untuk mengecek hasil kajian awal oleh tenaga keperawatan/kebidanan untuk sinkronisasi data pengisian, waspada jika terdapat masalah alergi obat pada pasien.

| Variable                                             | Cara Pengisian                                                                                                                                                                                                                       |
| ---------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Waktu Pengisian                                      | Pilih tanggal dan jam pengisian formulir                                                                                                                                                                                             |
| Riwayat Penyakit                                     | Diisi riwayat penyakit pasien. Contoh : pasien memiliki riwayat Diabetes Melitus Tipe 2 dan Hipertensi                                                                                                                               |
| Riwayat Alergi                                       | Diisi riwayat alergi pasien. Contoh : Alergi Amoxicillin (reaksi gatal dan ruam)                                                                                                                                                     |
| Masalah                                              | Diisi sesuai masalah kondisi pasien. Contoh : Pasien mengeluh mual saat mengonsumsi metformin                                                                                                                                        |
| Nama Obat                                            | Diisi sesuai nama obat yang diberikan kepada pasien. Contoh : Sulfonilurea                                                                                                                                                           |
| Jumlah                                               | Diisi sesuai jumlah obat yang diberikan kepada pasien. Contoh : 10 tablet                                                                                                                                                            |
| Dosis                                                | Diisi sesuai dosis obat yang diberikan kepada pasien. Contoh :  500 gr                                                                                                                                                               |
| Rute                                                 | Diisi sesuai rute obat yang diberikan kepada pasien. Contoh : Oral (diminum)                                                                                                                                                         |    
| Frekuensi                                            | Diisi sesuai frekuensi obat yang diberikan kepada pasien. Contoh : 3 x 1                                                                                                                                                             |
| Catatan                                              | Diisi catatan tambahan yang diperlukan. Contoh : Diminum segera sesudah makan                                                                                                                                                        | 
| A/T/P                                                | Pilih yang sesuai : Admisi ( Dipilih apabilaobat dikonsumsi saat pasien baru masuk); Transfer ( Dipilih jika terdapat perubahan instruksi obat saat pasien pindah); Pulang (Dipilih untuk obat yang diresepkan kepada pasien pulang) |          
| Rencana Awal                                         | Diisi dengan rencana tindakan farmasi yang akan dilakukan untuk memastikan terapi obat aman, efektif, dan sesuai kebutuhan pasien. Contoh : Pantau kadar glukosa darah sewaktu (GDS) dan tanda-tanda hipoglikemia setiap 8 jam       |                                                                                  |
