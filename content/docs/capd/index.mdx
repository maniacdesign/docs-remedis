| Variable                                  | Cara Pengisian                                                                                                                                     |
| ----------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------        |
| Nama Pasien                               | Nama lengkap sesuai kartu berobat/KTP, Contoh : Tn. Ahmad Fauzi                                                                                    |
| Usia                                      | Masukkan usia pasien saat ini, Contoh : 45 Tahun.                                                                                                  |
| Alamat                                    | Masukkan alamat domisili lengkap (untuk data medis), Contoh : Jl. RE Martadinata No. 15, Kel. Kotakulon, Kec. Bondowoso, Kab. Bondowoso.           |
| Nomor Telepon                             | Masukkan nomor aktif yang bisa dihubungi (WhatsApp/Telepon), Contoh : 0812-xxxx-5566.                                                              |
| Alamat Kirim                              | Pilih lokasi tujuan pengiriman logistik sesuai dengan KTP atau alamat lainnya                                                                      |
| Jenis                                     | Pilih Consumable atau Transfer Set sesuai kebutuhan pasien                                                                                         |
