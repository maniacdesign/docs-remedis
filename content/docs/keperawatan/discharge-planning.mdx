| Variable                                             | Cara Pengisian                                                                                                                                     |
| ---------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------        |
| Waktu Pengisian                                      | Pilih tanggal dan jam pengisian formulir                                                                                                           |
| Apakah Ada Kriteria Discharge Planning pada Pasien ? | Dipilih Ya jika pasien memenuhi setidaknya satu kriteria di bawahnya, atau Tidak jika tidak ada kriteria yang dipenuhi.                            |
| Umur > 65 Tahun                                      | Dipilih Ya jika usia pasien lebih dari 65 tahun.                                                                                                   |
| Keterbatasan Mobilisasi / Gerak                      | Dipilih Ya jika pasien memiliki keterbatasan signifikan dalam bergerak atau mobilisasi.                                                            |
| Kebutuhan Pelayanan Medis atau Perawat Lanjutan      | Dipilih Ya jika pasien diprediksi memerlukan perawatan atau prosedur medis/keperawatan khusus setelah pulang (misal: perawatan luka, fisioterapi). |
| Bantuan untuk Melakukan Aktifitas Sehari - hari      | Dipilih Ya jika pasien membutuhkan bantuan orang lain untuk melakukan aktivitas dasar sehari-hari (mandi, makan, berpakaian, dll.).                |
| Penanganan                                           | Dipilih jika ada kriteria discharge planing pada penanganan sesuai dengan kondisi pasien.                                                          |                                                                                                  |
