## 1. Cara Pengisian Skrining Gizi Dewasa dan Obstetri/Kehamilan/Nifas

<img
  src="/skrining_gizi_dewasa.png"
  alt="Contoh tampilan form pengkajian awal medis"
  width="750"
  style={{
    display: "block",
    margin: "0 auto",
    marginButtom: "20px" /* Ini yang membuat gambar rata tengah */,
    border: "2px solid blue",
  }}
/>


| Variable                  | Cara Pengisian                                                                                                                 |
| ------------------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| Parameter Berat Badan     | Pilih 'ya' apabila pasien pernah mengalami penurunan/kenaikan berat badan, tapi pilih 'tidak' bila hal tersebut tidak terjadi  |                                                        
| Parameter Asupan Makan    | Pilih 'ya' apabila pasien mengalami penurunan nafsu makan, pilih 'tidak' bila asupan pasien tidak menurun                      |
| Kondisi Khusus Pasien     | Pilih kondisi khusus Diabetes/Kemoterapi/Hemodialisa/Geriatri/Imunitas menurun                                                 | 
| Total                     | Berisi hasil nilai dari pengisian parameter                                                                                    |
| Kesimpulan                | Besisi kesimpulan apakah pasien mengalami malnutrisi atau tidak                                                                | 


## Cara Pengisian Skrining Gizi Anak

<img
  src="/skrining_gizi_anak.png"
  alt="Contoh tampilan form pengkajian awal medis"
  width="750"
  style={{
    display: "block",
    margin: "0 auto",

    border: "2px solid blue",
  }}
/>

| Variable                  | Cara Pengisian                                                                                                                 |
| ------------------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| Parameter Tubuh           | Pilih 'ya' apabila pasien anak tampak kurus, tapi pilih 'tidak' bila pasien anak tidak tampak kurus                            |                                                        
| Parameter Berat Badan     | Pilih 'ya' apabila pasien mengalami penurunan berat badan, pilih 'tidak' bila tidak mengalami penurunan                        |
| Kondisi Pasien            | Pilih kondisi yang sudah dialami pasien Muntah>3kali/Diare>5kali/                                                              | 
| Parameter Malnutrisi      | Pilih 'ya' apabila pasien mengalami malnutrisi, tapi pilih 'tidak' apabila pasien tidak mengalami malnutrisi                   |
| Kondisi Khusus Pasien     | Pilih salah satu Diare kronis/HIV/PJB/Hepatoma/Ginjal/Stoma                                                                    |
