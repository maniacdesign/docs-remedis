| Variable                         | Cara Pengisian                                                                                                             |
| -------------------------------- | -------------------------------------------------------------------------------------------------------------------------- |
| Waktu Pengisian                  | Diisi dengan tanggal dan waktu pengisian                                                                                   |
| Metode pengkajian                | Pilih metode Morse Fall Scale/Humpy Dumpty Fall Scale                                                                      |
| Riwayat Jatuh                    | Pilih 'tidak' bila tidak ada riwayat jatuh, pilih 'ya' bila ada riwayat jatuh                                              |
| Status Mental                    | Pilih status mental sesuai kondisi pasien                                                                                  |
| Efek Samping Obat                | Pilih 'tidak' apabila tidak ada efek samping obat, pilih 'ya' apabila ada efek samping dan isi keadaan efek samping obat   |
| Post GA/RA                       | Pilih sesuai kondisi pasien                                                                                                |
| Kelemahan dalam berjalan         | Pilih kelemahan pasien dalam berjalan                                                                                      |
| Kerusakan berjalan               | Pilih sesuai kondisi pasien                                                                                                |
| Alat Bantu                       | Pilih alat bantu yang digunakan pasien dalam berjalan                                                                      |
| Penyakit Penyerta                | Pilih 'tidak' bila pasien tidak memiliki penyakit penyerta, pilih 'ya' apabila pasien memiliki penyakit penyerta           |
| Terapi Intervena                 | PIlih sesuai kondisi pasien                                                                                                |
