| Variable     | Cara Pengisian                                                                                                                                                                                         |
| ------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Pilih pasien | Pilih pasien sebelum mengisi kajian gizi (Dewasa/Anak/Bayi)                                                                                                                                            |
| Antropometri | Bagian ini berisi pengisian pengukuran tubuh seperti BB, TB, BBI, IMT, dll. Pengisian ini digunakan untuk menentukan status gizi pasien                                                                |
| Biokimia     | Bagian ini digunakan mencatat hasil pemeriksaan laboratorium (seperti albumin, hemoglobin, atau gula darah). 'normal' apabila hasil pemeriksaan masih wajar, 'masalah' apabila terdapat hasil abnormal |
| Fisik/Klinis | Pilih kondisi fisik/klinis dan sebutkan keadaan fisik/klinis                                                                                                                                           |

| Variable               | Cara Pengisian                                                                                                                                                                                         |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Alergi Makanan Dahulu  | Pilih pasien sebelum mengisi kajian gizi (Dewasa/Anak/Bayi)                                                                                                                                            |
| Pola Makan Dahulu      | Bagian ini berisi pengisian pengukuran tubuh seperti BB, TB, BBI, IMT, dll. Pengisian ini digunakan untuk menentukan status gizi pasien                                                                |
| Susunan Makanan Dahulu | Bagian ini digunakan mencatat hasil pemeriksaan laboratorium (seperti albumin, hemoglobin, atau gula darah). 'normal' apabila hasil pemeriksaan masih wajar, 'masalah' apabila terdapat hasil abnormal |
| Dahulu Lainnya         | Pilih kondisi fisik/klinis dan sebutkan keadaan fisik/klinis                                                                                                                                           |
| Nafsu Makan Sekarang   | Pilih nafsu makan pasien sekarang (Baik/Kurang)                                                                                                                                                        |
| Kendala Makan Sekarang | Pilih salah satu kendala pasien saat makan (Sulit Makan/Sulit Mengunyah/Mual/Muntah)                                                                                                                   |
| Diet                   | Pilih jenis Diet pasien                                                                                                                                                                                |
