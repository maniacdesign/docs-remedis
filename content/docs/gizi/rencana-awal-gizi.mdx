## Cara Pengisian

<img
  src="/rencana_gizi.png"
  alt="Contoh tampilan form pengkajian awal medis"
  width="750"
  style={{ border: "2px solid blue" }}
/>

| Variable           | Cara Pengisian                                                       |
| ------------------ | -------------------------------------------------------------------- |
| Masalah Gizi       | Pilih tidak ada masalah gizi jika tidak ada masalah pada gizi pasien |
| Status Gizi        | Pilih status gizi dan sebutkan keadaan status gizi                   |
| Nilai Laboratorium | Pilih nilai laboratorium dan sebutkan keadaan nilai laboratorium     |
| Fisik/Klinis       | Pilih kondisi fisik/klinis dan sebutkan keadaan fisik/klinis         |
| Asupan Makanan     | Pilih asupan makanan untuk mengisi keadaan asupan makanan            |
| Diagnosa Gizi      | Diisi dengan diagnosa gizi yang diberikan ke pasien                  |
| Monitoring         | Pilih monitoring dan evaluasi pasien                                 |
| Edukasi            | Diisi dengan edukasi yang diberikan kepada Pasien                    |

| Variable                         | Cara Pengisian                                                                                                                                                                                                                                                                                             |
| -------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------  |
| Masalah Gizi                     | Pilih tidak ada masalah gizi                                                                                                                                                                                                                                                                               |
| Status Gizi                      | Pilih status gizi dan sebutkan keadaan status gizi, Contoh : Gizi Kurang (Underweight)                                                                                                                                                                                                                     |
| Nilai Laboratorium               | Pilih nilai laboratorium dan sebutkan keadaan nilai laboratorium, Contoh : Albumin: 2.8 g/dL (Rendah), Hb: 10.2 g/dL                                                                                                                                                                                       |
| Fisik/Klinis                     | Pilih kondisi fisik/klinis dan sebutkan keadaan fisik/klinis, Contoh : Oedema pada ekstremitas bawah, konjungtiva pucat.                                                                                                                                                                                   |
| Asupan Makanan                   | Pilih asupan makanan untuk mengisi keadaan asupan makanan, Contoh : Asupan menurun &lt; 50% dari porsi biasanya selama 1 minggu.                                                                                                                                                                               |
| Intervensi Gizi                  | Melakukan intervensi gizi dengan memberikan (Nama Diet & Bentuk Makanan), yang mengandung kebutuhan energi sebesar (Jumlah) kkal, protein (Jumlah) gram, lemak (Jumlah) gram, dan karbohidrat (Jumlah) gram, disertai instruksi khusus (Lainnya, misal: batasan cairan atau frekuensi makan).             |
| Diagnosa Gizi                    | Diisi dengan diagnosa gizi yang diberikan ke pasien, Contoh : (P) Kelebihan asupan cairan (E) berkaitan dengan kurangnya kepatuhan pembatasan asupan cairan harian (S) ditandai dengan adanya bengkak (oedema) pada kaki dan kenaikan berat badan >3 kg dalam seminggu.                                    |
| Monitoring                       | Pilih monitoring dan evaluasi pasien                                                                                                                                                                                                                                                                       |
| Edukasi                          | Diisi dengan edukasi yang diberikan kepada Pasien                                                                                                                                                                                                                                                          |
