### 1. Pengisian Tanggal Keluar, Diagnosis, & Tindakan
<img
  src="/cara_keluar.png"
  alt="Contoh tampilan form asuhan farmasi"
  width="750"
  style={{
    display: "block",
    border: "2px solid blue",
  }}
/>

| Variable                                             | Cara Pengisian                                                                 |
| ---------------------------------------------------- | -----------------------------------------------------------------------------  |
| Tanggal KRS                                          | Pilih tanggal sesuai tanggal pulang pasien                                     |
| Indikasi Dirawat                                     | Berisi ringkasan kondisi klinis atau keluhan utama pasien                      |
| Diagnosa Masuk                                       | Berisi diagnosis awal berdasarkan hasil pemeriksaan awal dokter                |
| Jenis Diagnosa                                       | Pilih jenis diagnosa apakah diagnosa pasien tergolong jenis Primer/Sekunder    |
| Kode ICD-10                                          | Diisi kode ICD - 10 yang sesuai dengan diagnosa dokter                         |
| Kasus                                                | Pilih jenis kasus perawatan pasien, apakah tergolong kasus Lama/Baru           |
| Jenis Tindakan                                       | Pilih Klasifikasi jenis tindakan medis yang dilakukan (Utama/Lainnya )         |
| Kode ICD-9                                           | Diisi kode ICD - 9 sesuai dengan tindakan atau prosedur yang dilakukan         |

| Variable                         | Cara Pengisian                                                             |
| -------------------------------- | -------------------------------------------------------------------------  |
| Diit                             | Diisi dengan jenis diet pasien sesuai kondisi klinis                       |
| Farmakologi                      | Diisi sesuai dengan obat-obatan yang diberikan kepada pasien               |
| Terapi Lain                      | Diisi sesuai dengan terapi non-obat yang dilakukan                         |
| Hasil Pemeriksaan Penting        | Diisi dengan hasil pemeriksaan penunjang yang bermakna                     |
| Konsultasi                       | Diisi jika ada konsultasi dengan bagian lain                               |
| Instruksi                        | Diisi dengan instruksi medis/tindak lanjut untuk tenaga kesehatan          |
| Prognosa                         | Diisi dengan perkiraan kondisi pasien ke depan                             |
| Kondisi saat keluar              | Pilih salah satu kondisi yang sesuai dengan kondisi pasien saat pulang     |
| Follow up                        | Pilih rencana tindak lanjut pasien setelah perawatan/transfer              |
