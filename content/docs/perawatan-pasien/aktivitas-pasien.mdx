## Cara Pengisian Form Aktivitas Pasien

<img
  src="/aktivitaspasien1.png"

  width="750"
  style={{
    display: "block",

  }}
/>


| Variable                                             | Cara Pengisian                                                                                                                                     |
| ---------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------        |
| Tanggal Pengisian                                    | Diisi sesuai tanggal pengisian aktivitas harian pasien                                                                                             |
| Kecepatan Awal                                       | Diisi kecepetan awal aktivitas saat pasien mulai latihan.                                                                                          |
| Max Workload                                         | Diisi dengan beban kerja maksimal yang dapat dicapai pasien pada saat latihan atau evaluasi                                                        |
| Target Kecepatan                                     | Diisi dengan dengan kecepatan target yang ingin dicapai dalam rencana aktivitas atau latihan pasien.                                               |
| Kategori Aktivitas                                   | Pilih kategori aktivitas fisik pasien sesuai tingkat intensitas kegiatan sehari-hari                                                               |
