| Variable                      | Cara Pengisian                                                                                                |
| ----------------------------- | ------------------------------------------------------------------------------------------------------------- |
| Waktu Pengisian               | Diisi dengan tanggal dan waktu pengisian assesmen ulang nyerin                                                |
| Skor Nyeri                    | Pilih skor nyeri yang sesuai kondisi pasien                                                                   |
| Skor Sedasi                   | Pilih skor sedasi yang sesuai kondisi pasien                                                                  |
| Pengkajian Selanjutnya        | Waktu spesifik saat obat harus diberikan, Contoh : 08.00, 16.00, 24.00                                        |
| Intervensi Non-Farmakologi    | Pilih Intervensi Non - Farmakologi sesuai kondisi                                                             |
| Nama Obat                     | Diisi sesuai nama obat yang diberikan pada pasien, Contoh : Ceftriaxone atau Furosemide                       |
| Dosis dan Frekuensi           | Diisi sesuai dosis dan frekuensi pemberian obat yang diberikan pada pasien, Contoh : 1 gram per 12 jam (2x1)  |
| Rute                          | Diisi rute pemberian obat yang diberikan pada pasien, Contoh : IV (Intravena) / Oral / IM / OGT               |
