## Cara Pengisian Formulir Edukasi Pasien dan Keluarga Terintegrasi


#### 1. Persiapan Edukasi
<img
  src="/edukasi_keluarga1.png"
  alt="Contoh tampilan form pengkajian awal medis"
  width="750"
  style={{
    display: "block",
    margin: "0 auto",
    marginButtom: "20px" /* Ini yang membuat gambar rata tengah */,
    border: "2px solid blue",
  }}
/>

| Variable                                                  | Cara Pengisian                                                                                          |
| --------------------------------------------------------- | ------------------------------------------------------------------------------------------------------  |
| Waktu Pengisian                                           | Diisi dengan tanggal dan jam saat asesmen kebutuhan edukasi dilakukan.                                  |
| Bahasa                                                    | Dipilih bahasa utama yang digunakan pasien. Jika Suku Bangsa atau Lain-lain, sebutkan bahasanya.        |
| Kebutuhan Penerjemah                                      | Dipilih Ya jika bahasa pasien tidak dikuasai oleh petugas atau dibutuhkan penerjemah profesional        |
| Pendidikan Pasien                                         | Dipilih tingkat pendidikan formal terakhir yang ditamatkan pasien.                                      |
| Baca dan Tulis                                            | Dinilai kemampuan pasien dalam membaca dan menulis informasi                                            |
| Tipe Pembelajaran                                         | Dipilih tipe pembelajaran yang paling efektif untuk pasien (Verbal: mendengarkan, Tulisan: membaca)     |
| Hambatan Edukasi                                          | Dipilih/dicentang semua faktor yang menjadi penghambat dalam pemberian edukasi kepada pasien.           |
| Kesediaan Menerima Edukasi                                | Dipilih kesediaan pasien untuk menerima informasi dan edukasi.                                          |


#### 2. Hak dan Kewajiban Pasien dan Keluarga (Perawat / Bidan)
<img
  src="/edukasi_keluarga2.png"

| Variable                                                  | Cara Pengisian                                                                                              |
| --------------------------------------------------------- | ------------------------------------------------------------------------------------------------------      |
| Tanggal Edukasi                                           | Tanggal Re-Edukasi                                                                                          |
| Durasi Waktu (Menit)                                      | Diisi lama waktu edukasi yang diberikan (dalam menit).                                                      |
| Sasaran Edukasi                                           | Dipilih pihak yang menerima edukasi, pasien atau keluarga pasien.                                           |
| Lainnya                                                   | Diisi jika edukasi diberikan kepada pihak lain (misal: penanggung jawab, pengantar).                        |
| Tingkat Pemahaman Awal                                    | Dipilih tingkat pengetahuan pasien/keluarga terhadap materi hak dan kewajiban sebelum edukasi diberikan     |
| Metode Edukasi                                            | Dipilih metode yang digunakan saat menyampaikan edukasi                                                     |
| Sarana Edukasi                                            | Dipilih media/sarana yang digunakan untuk membantu edukasi                                                  |
| Evaluasi                                                  | Digunakan untuk menilai keberhasilan edukasi.                                                               |
| Sudah Mengerti                                            | Dipilih jika pasien/keluarga menunjukkan pemahaman yang memadai.                                            |
| Tanggal Re-Edukasi                                        | Diisi tanggal saat edukasi diulang atau diperkuat, jika diperlukan.                                         |
