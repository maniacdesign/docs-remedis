    ## Tampilan Pengisian CPPT Dokter 
  ### 1. CPPT SOAP
  <img
  src="/cppt_soap_dokter.png"
  alt="Contoh tampilan form pengkajian awal medis"

| Variable                         | Cara Pengisian                                                                                                                                                                                                                                       |
| -------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Subjektif                        | Diisi dengan data yang diperoleh dari keluhan pasien atau apa yang pasien rasakan dan katakan. Contoh : Pasien mengeluh nyeri perut kanan bawah, dirasakan tajam, hilang timbul sejak 2 jam lalu                                                     |                                                                                         |
| Objektif                         | Diisi dengan hasil pemeriksaan pasien seperti suhu tubuh, tekanan darah, respiratory rate, dll. Contoh : KU: Tampak sakit sedang. Thorax: Vesikuler +/+, Rhonki -/-, Wheezing -/-. Abdomen: Distensi (-), BU (+) normal. Lab: Hb 10.2, WBC 12.000    |                                                                                        |
| Assesmen                         | Diisi dengan diagnosis medis, diagnosis keperawatan, atau status perkembangan kondisi pasien (apakah membaik, tetap, atau memburuk). Contoh : Nyeri akut belum teratasi, gangguan mobilitas fisik.                                                   |
| Plan                             | Diisi dengan rencana tindakan yang akan dilakukan untuk menangani masalah pasien. Contoh : Observasi TTV, ajarkan teknik relaksasi, bantu mobilisasi miring                                                                                          |                                                                                                                                                           |

| Variable                         | Cara Pengisian                                                                                                                                                                                                                                                                                                               |
| -------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Situasi                          | Diisi sesuai kondisi pasien terkini. Contoh : Pasien mengalami sesak napas berat secara tiba-tiba disertai penurunan saturasi oksigen hingga 88% dengan nasal canula 4 lpm                                                                                                                                                   |                                                                                                                                                   |
| Latar Belakang                   | Diisi dengan informasi klinis yang relevan dengan situasi saat ini untuk memberikan konteks. Contoh : Pasien masuk 2 jam lalu dengan diagnosis gagal jantung kongestif (CHF). Riwayat hipertensi tidak terkontrol. Pasien sudah mendapatkan injeksi Furosemide 20mg di IGD namun urine output masih minimal                  |
| Assesmen                         | Diisi dengan diagnosis medis, diagnosis keperawatan, atau status perkembangan kondisi pasien (apakah membaik, tetap, atau memburuk). Contoh : Kesadaran Compos Mentis, TD: 170/100 mmHg, HR: 110x/menit, RR: 28x/menit. Terdapat suara ronkhi basah halus di kedua lapang paru bawah. Saya mencurigai adanya edema paru akut |
| Rekomendasi                      | Diisi sesuai rencana tindakan yang akan dilakukan. Contoh : Observasi ulang frekuensi napas dan suara mengi 15 menit setelah nebulisasi. Jika tidak membaik, lapor kembali untuk pertimbangan rawat ICU                                                                                                                      |
