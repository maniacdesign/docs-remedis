## Cara Pengisian Form Catatan Implementasi Pasien Terintegrasi

#### 1. Tindakan
<img
  src="/catatanimplementasi.png"
  alt="Contoh tampilan form Catatan Implementasi"
  width="750"
  style={{
    display: "block",
    border: "2px solid blue",
  }}
/>


| Variable                                             | Cara Pengisian                                                                                                                                                      |
| ---------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------        |
| Tanggal                                              | Diisi tanggal dan jam saat tindakan dilakukan.                                                                                                                      |
| Implementasi                                         | Diisi deskripsi tindakan keperawatan atau intervensi PPA lain yang telah dilaksanakan (misal: Mobilisasi miring kanan, Pemberian edukasi diet, Pemasangan infus)    |
| Pelaksana                                            | Diisi nama atau inisial petugas PPA yang melaksanakan tindakan                                                                                                      |
| Aksi                                                 | Tombol untuk menambah atau mengubah baris data Tindakan                                                                                                             |

#### 2. Obat
<img
  src="/catatanimplementasi2.png"
  alt="Contoh tampilan form Catatan Implementasi"
  width="750"
  style={{
    display: "block",
    border: "2px solid blue",
  }}
/>


| Variable                                          | Cara Pengisian                                                                                         |
| ------------------------------------------------- | -----------------------------------------------------------------------------------------------        |
| Tanggal                                           | Diisi tanggal dan jam saat tindakan dilakukan.                                                         |
| Implementasi                                      | Diisi nama obat, dosis, rute, dan waktu pemberian (misal: Paracetamol 500mg, oral, jam 10.00).         |
| Aksi                                              | Tombol untuk menambah atau mengubah baris data Obat.                                                   |
