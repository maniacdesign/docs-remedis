| Variable                                             | Cara Pengisian                                                                                        |
| ---------------------------------------------------- | ----------------------------------------------------------------------------------------------------- |
| Tanggal Pemberian                                    | Pilih tanggal dan jam pengisian formulir. Contoh : 12/31/2025 08:43 AM                                |
| Nama Obat                                            | Diisi nama obat yang akan diberikan. Contoh : Ceftriaxone atau Ondansetron                            |
| Jumlah                                               | Diisi jumlah obat yang akan diberikan. Contoh : 2 tablet                                              |
| Dosis                                                | Diisi dosis obat yang akan diberikan.  Contoh : 500mg                                                 |
| Rute                                                 | Diisi rute pemberian obat. Contoh : Oral                                                              |
| Frekuensi                                            | Diisi frekuensi pemberian obat. Contoh : 2 x 1                                                        |
