##### 1. Subject
| Variable                         | Cara Pengisian                                                                                    |
| -------------------------------- | --------------------------------------------------------------------------------------------------|
| Keluhan                          | Diisi keluhan yang dirasakan pasien saat datang ke Rumah Sakit                                    |
| Riwayat Penyakit Sekarang        | Diisi penyakit yang diderita pasien saat ini                                                      |
| Riwayat Penyakit Dahulu          | Diisi riwayat penyakit pasien terdahulu                                                           |
| Riwayat Penyakit Keluarga        | Diisi riwayat penyakit keluarga                                                                   |  



##### 2. Objective
| Variable                         | Cara Pengisian                                                                                    |
| -------------------------------- | --------------------------------------------------------------------------------------------------|
| Keadaan umum                     | Diisi kondisi umum pasien                                                                         |
| Kesadaran                        | Diisi kondisi kesadaran pasien                                                                    |
| Tekanan Darah (TD)               | Diisi nilai tekanan darah pasien                                                                  |
| Suhu                             | Diisi suhu pasien                                                                                 |  
| Nadi                             | Diisi nadi pasien                                                                                 |
| Respiratory Rate                 | Diisi nilai respiratory rate pasien                                                               |
| Berat Badan                      | Diisi berat badan pasien                                                                          |
| Tinggi Badan                     | Diisi tinggi badan pasien                                                                         |
| Catatan                          | Diisi kondisi lain pasien yang belum tertulis diatas                                              |

##### 3. Assesmen
| Variable                         | Cara Pengisian                                                                                    |
| -------------------------------- | --------------------------------------------------------------------------------------------------|
| Diagnosa                         | Klik menu “tambah” untuk menambahkan diagnosa                                                     |
| ICD 9 CM                         | Klik menu “tambah” untuk menambahkan tindakan                                                     |


##### 4. Plan
| Variable                         | Cara Pengisian                                                                                    |
| -------------------------------- | --------------------------------------------------------------------------------------------------|
| Laboratorium                     | Klik menu “tambah” untuk menambahkan pemeriksaan laboratorium pada pasien                         |
| Radiologi                        | Klik menu “tambah” untuk menambahkan pemeriksaan radiologi pada pasien                            |
| Resep                            | Klik menu “tambah” untuk menambahkan resep untuk pasien                                           |
| Planning                         | Diisi rencana pemeriksaan yang belum tertulis                                                     |
