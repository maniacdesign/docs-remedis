| Variable                         | Cara Pengisian                                                                                                                                                                                    |
| -------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Alergi                           | Diisi dengan jenis alergi seperti obat, makanan, dll. Apabila tidak ada tulis "tidak ada" atau " - "                                                                                              |
| Keluhan utama                    | Diisi dengan keluhan yang paling dirasakan pasien saat berkunjung berobat. Contoh : Nyeri pinggang kiri menjalar ke kaki sejak 4 hari lalu                                                        |
| Riwayat penyakit sekarang        | Diisi dengan detail penjelasan mengenai awal mula, durasi, frekuensi, dan kronologi keluhan utama. Contoh : Nyeri muncul setelah mengangkat beban berat (kecetit), nyeri berkurang saat berbaring |                                   
| Pemeriksaan fisik                | Diisi dengan hasil pemeriksaan objektif. Contohnya : pemeriksaan tanda vital, NRS/Skala Nyeri, tes spesifik seperti SLR, palpasi otot, dan lain - lain                                            |                                                                                           
| Kesimpulan                       | Diisi dengan dengan kesimpulan hasil pemeriksaan seperti diagnosis medis atau suspek (dugaan) penyakit berdasarkan hasil pemeriksaan                                                              |                              
| Rencana uji fungsi               | Diisi dengan tindakan uji fungsi spesifik yang akan dilakukan untuk menilai keterbatasan fisik pasien. Contoh : Uji fungsi jalan, Tes lingkup gerak sendi (LGS) lumbal.                           |                                                                 
| Pemeriksaan lain                 | Diisi dengan catatan tambahan seperti hasil laboratorium, Rontgen, MRI, atau pemeriksaan penunjang lainny                                                                                         |
| Program                          | Diisi dengan rencana terapi rehabilitasi medik yang diberikan (seperti fisioterapi, okupasi terapi, dll).                                                                                         |
  </Tab>

| Variable                            | Cara Pengisian                                                                                                                                                                                                  |
| ----------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| KFR                                 | Diisi dengan nama dokter spesialis Kedokteran Fisik dan Rehabilitasi yang menangani                                                                                                                             |
| Koding                              | Diisi dengan kode prosedur tindakan sesuai standar ICD-9-CM. Contoh : 93.39 (Other physical therapy)                                                                                                            |
| Jenis Pemeriksaan                   | Diisi dengan nama tes atau prosedur uji fungsi yang dilakukan. Contoh : Pemeriksaan lingkup gerak sendi / Tes kekuatan otot                                                                                     |                                   
| Diagnosa fungsional                 | Diisi dengan hambatan fungsi tubuh atau gangguan aktivitas yang dialami pasien. Contoh : Gangguan nyeri                                                                                                         |                                                                                           
| Diagnosa medis                      | Diisi dengan diagnosis klinis atau penyakit yang mendasari keluhan. Contoh : LBP susp HNP                                                                                                                       |                              
| Instrumen Uji fungsi /prosedur KFR  | Diisi dengan catatan tambahan seperti hasil laboratorium, Rontgen, MRI, atau pemeriksaan penunjang lainny                                                                                                       |
| Hasil yang di dapat                 | Diisi dengan temuan klinis detail setelah dilakukan pengujian. Contoh : NRS 3-4 SLR -/- spasme Parathoracolumbal +/++ hiperestesi area plantar pedis S                                                          |
| Kesimpulan                          | Diisi dengan ringkasan akhir kondisi fungsional pasien setelah pemeriksaan. Contoh : Gangguan nyeri                                                                                                             |
| Rekomendasi                         | Diisi dengan rencana terapi, penggunaan alat bantu, atau latihan mandiri yang disarankan. Contoh : TENS IF 4000 mid parathoracolumbal dan IRR a/r mid parathoracolumbal, LSO semi rigid, core strength exercise |
  </Tab>

| Variable                              | Cara Pengisian                                                                                                    |
| ------------------------------------- | ------------------------------------------------------------------------------------------------------------------|
| Subjektif                             | Diisi dengan keluhan utama pasien, lokasi nyeri, serta riwayat kejadian yang diceritakan oleh pasien              |                                                                                                                 
| Objektif                              | Diisi dengan hasil pemeriksaan fisik yang terukur, nilai ambang nyeri, serta tes provokasi yang dilakukan         |
| Assesmen                              | Diisi dengan penegakan diagnosis fungsional dan diagnosis medis berdasarkan temuan data sebelumnya.               |
| Goal of treatment                     | Diisi dengan target jangka pendek dan jangka panjang yang ingin dicapai dari terapi ini.                          |
| Tindakan / program rehabilitasi medik | Diisi dengan rincian modalitas alat, dosis, frekuensi, serta latihan fisik yang akan diberikan.                   |
| Edukasi                               | Diisi dengan instruksi khusus, saran aktivitas, atau larangan yang harus dipatuhi pasien di rumah                 |
| Frekuensi kunjungan                   | Diisi dengan jadwal rutin atau jumlah pertemuan terapi yang disarankan dalam satu periode                         |
| Rencana Tindak Lanjut                 | Diisi dengan langkah medis berikutnya setelah rangkaian terapi selesai dilakukan.                                 |
