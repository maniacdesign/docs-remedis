| Variable                         | Cara Pengisian                                                                                                                                                                     |
| -------------------------------- | -----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Pemeriksaan fisik                | Diisi dengan hasil temuan umum pada pemeriksaan fisik pasien secara menyeluruh. Contoh : keadaan umum pasien, kesadaran, dan tanda vital                                           |                                    
| Gambar Tenggorokkan              | Berikan tanda lingkaran pada gambar dengan klik "Coretan Tablet" untuk menandai area yang bermasalah. Contoh :  amandel yang membesar atau adanya bercak pada diagram anatomi      |                                                                          
| Catatan Tenggorokan              | Diisi deskripsikan temuan spesifik pada tenggorokan seperti kondisi uvula, tonsil (amandel), dan faring.                                                                           |
| Kepala                           | Diisi dengan hasil inspeksi dan palpasi pada bagian kepala secara umum. Contoh : Normocephalic, tidak ada massa, tidak ada nyeri tekan pada sinus                                  |
| Leher                            | Diisi dengan temuan pada area leher, seperti mobilitas atau adanya kaku kuduk.                                                                                                     |
| Pembesaran Kelenjar Getah Bening | Tentukan apakah ada pembengkakan pada kelenjar getah bening di area leher atau belakang telinga                                                                                    |                           
  </Tab>

| Variable                         | Cara Pengisian                                                                                                                                                                     |
| -------------------------------- | -----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Gambar Telinga                   | Berikan tanda lingkaran dengan klik "Coretan Tablet" untuk menandai letak kelainan secara visual pada diagram telinga.                                                             |
| Liang Telinga                    | Pilih kondisi liang telinga sesuai kondisi pasien. Kemudian tambahkan keterangan jika diperlukan                                                                                   |
| Membran Timpani                  | Pilih kondisi membran timpani sesuai kondisi pasien. Apabila pilihan kondisi tidak ada yang sesuai maka tambahkan pada bagian "lain - lain"                                        |
| Kavum Timpani                    | Pilih kondisi kavum timpani sesuai kondisi pasien. Apabila pilihan kondisi tidak ada yang sesuai maka tambahkan pada bagian "lain - lain"                                          |
| Lain - lain                      | Diisi apabila terdapat temuan tambahan yang tidak masuk kategori di atas. Contoh : kondisi belakang telinga/retroaurikuler                                                         |
| Kesimpulan                       | Diisi dengan diagnosis kerja atau ringkasan hasil pemeriksaan yang paling menonjol                                                                                                 |
| Anjuran                          | Diisi dengan instruksi untuk pasien atau rencana tindakan selanjutnya. Contoh : Jaga telinga agar tetap kering (jangan kemasukan air), kontrol 1 minggu lagi, minum obat rutin     |

  </Tab>

| Variable                         | Cara Pengisian                                                                                                                                                                     |

| Kavum Nasi                       | Pilih kondisi kavum nasi sesuai kondisi pasien                                                                                                                                     |
| Konka Inferior                   | Pilih kondisi konka inferior sesuai kondisi pasien                                                                                                                                 |
| Meatus Medius                    | Pilih kondisi meatus medius sesuai kondisi pasien                                                                                                                                  |
| Septum                           | Pilih kondisi septum sesuai kondisi pasien                                                                                                                                         |
| Nasofaring                       | Pilih kondisi nasofaring sesuai kondisi pasien                                                                                                                                     |
| Lain - lain                      | Diisi apabila terdapat temuan tambahan yang tidak masuk kategori di atas. Contoh : Post-Nasal Drip (+), darah mengalir di dinding faring, atau nyeri tekan sinus (+).              |
| Kesimpulan                       | Diisi dengan instruksi untuk pasien atau rencana tindakan selanjutnya. Contoh : Rhinitis Alergi dengan Septum Deviasi, atau Sinusitis Maksilaris Akut                              |
