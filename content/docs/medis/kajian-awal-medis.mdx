| Variable                         | Cara Pengisian                                                                                                                                                                                |
| -------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Waktu Pengisian                  | Pilih tanggal dan jam pengisian formulir                                                                                                                                                      |
| Keluhan Utama                    | Diisikan dengan keluhan utama                                                                                                                                                                 |
| Riwayat penyakit pasien sekarang | Diisikan dengan riwayat pasien saat ini atau kejadian terbaru yang dialami oleh pasien.                                                                                                       |
| Riwayat Penyakit                 | Diisikan dengan riwayat penyakit pasien terdahulu                                                                                                                                             |
| Alergi                           | Pilih jenis alergi - isi keterangan terkait alergi pasien - klik tambah                                                                                                                       |
| Pemberian Obat                   | Diisikan dengan nama obat, dosis, dan lama penggunaan - klik tambah                                                                                                                           |
| Perilaku Berisiko                | Pilih perilaku pasien yang beresiko dengan klik checkbox                                                                                                                                      |
| Perilaku Lain                    | Diisi perilaku lain pasien yang tidak disebutkan diatas                                                                                                                                       |




## 2. Cara Pengisian Pemeriksaan Fisik

<img 
  src="/kajianawal_fisik.png" 
  alt="Contoh tampilan form pengkajian awal medis" 
  width="750" 
  style={{ 
    display: 'block', 
    margin: '0 auto', /* Ini yang membuat gambar rata tengah */
    border: '2px solid blue' }}
/>


| Variable                         | Cara Pengisian                                                                                                                                                                                |
| -------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Skala GCS                        | Diisikan sesuai dengan tingkat kesadaran pasien berdasarkan respons mata, suara, dan gerak.                                                                                                   |
| Kesadaran                        | Pilih kesadaran pasien yang sesuai dengan kondisi pasien Composmentis/Apatis/Samnolen/Stupor/Coma                                                                                             |
| Nadi                             | Diisi dengan jumlah denyut nadi pasien dalam satu menit (x/menit)                                                                                                                             |
| Tensi                            | Diisi dengan hasil tekanan darah (tensi) pasien                                                                                                                                               |
| Suhu                             | Diisi dengan tingkat suhu pasien yang telah diperiksa                                                                                                                                         |
| Respiratory Rate                 | Diisi dengan jumlah frekuensi napas pasien dalam satu menit                                                                                                                                   |
| SPO2                             | Diisi dengan persentase saturasi oksigen dalam darah                                                                                                                                          |
| Tinggi Badan                     | Diisi dengan tinggi badan pasien                                                                                                                                                              |
| Berat Badan                      | Diisi dengan berat badan pasien                                                                                                                                                               |
| Bagian Tubuh                     | Pilih bagian tubuh yang terdapat keluhan seperti Kepala/Dada/Abdomen/Tulang Belakang/Genotalia/Ekstremitas                                                                                    |
