| Variable                         | Cara Pengisian                                                                                                                                                                                |
| -------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Waktu Pengisian                  | Pilih tanggal dan jam pengisian formulir                                                                                                                                                      |
| Keluhan Utama                    | Diisikan dengan keluhan utama pasien saat datang berobat. Contoh : pasien mengalami nyeri perut di bagian kanan bawah sejak 2 hari yang lalu                                                  |
| Riwayat penyakit pasien sekarang | Diisikan dengan kronologi keluhan utama. Contoh : Nyeri terasa tajam dan menusuk, memberat saat berjalan, awalnya nyeri di sekitar pusar lalu berpindah ke kanan bawah. Skala nyeri 7/10      |
| Riwayat Penyakit                 | Diisikan dengan riwayat penyakit pasien terdahulu. Contoh : Hipertensi terkontrol sejak 3 tahun lalu, pernah operasi usus buntu (jika relevan)                                                |
| Alergi                           | Pilih jenis alergi - isi keterangan terkait alergi pasien - klik tambah. Contoh : Alergi obat amoksisilin, reaksi berupa gatal dan ruam kemerahan setelah konsumsi obat                       |
| Pemberian Obat                   | Diisikan dengan nama obat, dosis, dan lama penggunaan - klik tambah. Contoh : Nama obat : Obat paracetamol, dosis : 50gr, Lama : 3x1 (setelah makan)                                          |
| Perilaku Berisiko                | Pilih perilaku pasien yang beresiko dengan klik checkbox                                                                                                                                      |
| Perilaku Lain                    | Diisi perilaku lain pasien yang tidak disebutkan diatas                                                                                                                                       |

| Variable                         | Cara Pengisian                                                                                                                                                                                |
| -------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Skala GCS                        | Diisikan sesuai dengan tingkat kesadaran pasien berdasarkan respons mata, suara, dan gerak.                                                                                                   |
| Kesadaran                        | Pilih kesadaran pasien yang sesuai dengan kondisi pasien Composmentis/Apatis/Samnolen/Stupor/Coma                                                                                             |
| Nadi                             | Diisi dengan jumlah denyut nadi pasien dalam satu menit (x/menit). Contoh : 80 /menit                                                                                                         |
| Tensi                            | Diisi dengan hasil tekanan darah (tensi) pasien. Contoh : 120/80 mmhg                                                                                                                         |
| Suhu                             | Diisi dengan tingkat suhu pasien yang telah diperiksa. Contoh : 38Â°C                                                                                                                          |
| Respiratory Rate                 | Diisi dengan jumlah frekuensi napas pasien dalam satu menit. Contoh : 20/menit                                                                                                                |
| SPO2                             | Diisi dengan persentase saturasi oksigen dalam darah. Contohnya : 98%                                                                                                                         |
| Tinggi Badan                     | Diisi dengan tinggi badan pasien                                                                                                                                                              |
| Berat Badan                      | Diisi dengan berat badan pasien                                                                                                                                                               |
| Bagian Tubuh                     | Pilih bagian tubuh yang terdapat keluhan seperti Kepala/Dada/Abdomen/Tulang Belakang/Genotalia/Ekstremitas                                                                                    |

| -------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Kode ICD-10                      | Cari Kode ICD-10 yang sesuai dengan diagnosis medis                                                                                                                                                                                                                                                                                |
| Diagnosis Medis                  | Diisi dengan diagnosis medis yang ditetapkan oleh dokter.  Contohnya : Gastroenteritis akut dengan dehidrasi ringan                                                                                                                                                                                                                |
| Permasalahan Medis               | Diisi dengan permasalahan medis sesuai dengan kondisi pasien. Contohnya : Nyeri perut kanan bawah sejak 2 hari lalu secara mendadak. Nyeri terasa terus-menerus dan tajam, durasi meningkat saat bergerak atau batuk, berkurang sedikit saat berbaring menekuk kaki. Disertai mual, muntah 1x, dan demam ringan. Skala nyeri 7/10. |
