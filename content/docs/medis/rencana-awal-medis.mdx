## Cara Pengisian Rencana Awal Medis

| Variable                       | Cara Pengisian                                                                                                                              |
| ------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------- |
| Waktu Pengisian                | Pilih tanggal dan jam pengisian formulir                                                                                                    |
| Diagnosa dan Masalah Medis     | Diisi dengan diagnosa dan masalah medis sesuai hasil pemeriksaan, Contoh : Gastroenteritis Akut dengan Dehidrasi Ringan-Sedang, Vomitus Profus.                                                                          |
| Sasaran dan Target Waktu       | Diisi hasil yang diharapkan dari perawatan, Contoh : Rehidrasi tercapai dan muntah berhenti dalam waktu 24 jam.                                                 |
| Diagnosa Banding               | Diisi diagnosis lain yang harus dipertimbangkan sebelum diagnosis akhir ditegakkan, Contoh : Demam Tifoid, Disentri Basiler, Appendisitis Akut.                                                         |
| Rencana Pemeriksaan Diagnostik | Diisi pemeriksaan lanjutan yang akan dilakukan, Contoh : tes darah, rontgen, CT scan.                                                     |
| Rencana Terapi                 | Diisi dengan rencana terapi yang perlu dilakukn pasien, Contoh : IVFD RL 20 tpm, Inj. Ondansetron 4mg/8 jam, Inj. Ranitidin 50mg/12 jam.                                                                                      |
| Rencana Monitoring             | Diisi dengan parameter yang perlu dipantau secara berkala, Contoh : Monitoring tanda vital per 4 jam, monitoring balans cairan, dan frekuensi BAB. |
| Rencana Edukasi                | Diisi dengan edukasi yang perlu diberikan pada pasien, Contoh : Edukasi mengenai penyebab diare, pentingnya asupan cairan, dan kebersihan tangan (hand hygiene).                                                                                       |

<Callout title="Waktu pengisian" type="warn">
  Perhatikan dalam pengisian "waktu pengisian" agar mengisi dengan waktu yang
  sesuai
</Callout>
