### 1. Riwayat Kelahiran
<img
  src="/neonatologi.png"
  alt="Contoh tampilan form pengkajian awal medis"
  width="750"
  style={{border: "2px solid blue",}}
/>

| Variable                         | Cara Pengisian                                                                                                              |
| -------------------------------- | --------------------------------------------------------------------------------------------------------------------------- |
| Nama Ibu                         | Diisi dengan nama ibu                                                                                                       |
| Usia Ibu                         | Diisi dengan usia ibu                                                                                                       |
| Kehamilan                        | Diisi dengan usia kehamilan pasien dalam satuan minggu. Contoh : 38 minggu                                                  |
| G (Gravida)                      | Diisi dengan jumlah total kehamilan yang pernah dialami pasien, termasuk kehamilan saat ini. Contoh : G : 1                 |
| P (Paritas)                      | Diisi dengan jumlah seluruh kelahiran cukup bulan yang pernah dialami ibu. Contoh : P : 0                                   |
| Yang Menolong Partus             | Diisi sesuai petugas kesehatan yang menolong persalinan                                                                     |
| Komplikasi Kehamilan             | Diisi dengan komplikasi kehamilan yang dialami pasien. Contoh : Pada usia kandungan 28 minggu pasien menngalami preclampsia |
| Tanggal Lahir                    | Diisi dengan tanggal kelahiran bayi pasien                                                                                  |
| Cara Lahir                       | Pilih jenis tindakan sesuai cara lahir bayi                                                                                 |
| Ketuban                          | Pilih kondisi ketubah                                                                                                       |

| Variable                         | Cara Pengisian                                                                                                                           |
| -------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| Jenis Kelamin Bayi               | Pilih jenis kelamin bayi                                                                                                                 |
| Berat Bayi                       | Diisi sesuai hasil pengukuran berat bayi. Contoh : 3200gr                                                                                |
| Panjang Bayi                     | Diisi sesuai hasil pengukuran panjang bayi. Contoh : 49 cm                                                                               |
| Lingkar Dada (LD)                | Diisi sesuai hasil pengukuran lingkar dada bayi. Contoh : 33 cm                                                                          |
| Lingkar Kepala (LK)              | Diisi sesuai hasil pengukuran lingkar kepala bayi. Contoh : 35 cm                                                                        |
| Kembar/Tunggal, Lahir di         | Diisi dengan jenis kelahiran bayi (tunggal/kembar), kemudian tambakan lokasi bayi. Contoh : lahir Tunggal, Lahir di RS (Kamar Bersalin)  |
| Keadaan Saat Lahir               | Pilih keadaan bayi saat lahir                                                                                                            |
| Keterangan Kematian              | Pilih keterangan kematian apabila keaadaan bayi meninggal                                                                                |
| Cara Lahir                       | Pilih jenis tindakan sesuai cara lahir bayi                                                                                              |
| Sebab Kematian                   | Diisi sebab kematian bayi apabila keaadaan bayi meninggal. Contoh : bayi mengalami asfiksia berat                                        |
