## Cara Pengisian Assesmen Nyeri Neonatus

<img
  src="/assesment_nyeri_neonatus.png"
  alt="Contoh tampilan form pengkajian awal medis"
  width="750"
  style={{border: "2px solid blue",}}
/>


| Variable                         | Cara Pengisian                                                                                    |
| -------------------------------- | ------------------------------------------------------------------------------------------------- |
| Tanggal dan Waktu Pengisian      | Diisi dengan tanggal dan waktu pengisian formulir assesmen nyeri neonatus                         |
| Ekspresi Wajah                   | Pilih ekspresi wajah sesuai kondisi bayi saat lahir (Santa/Meringis)                              |
| Tangisan                         | Pilih jenis tangisan sesuai dengan kondisi bayi saat lahir (Tidak Menangis/Merengek/Menangis)     |
| Pola Pernafasan                  | Pilih pola pernafasan bayi saat baru lahir (Teratur/Berubah)                                      |
| Lengan                           | Pilih pergerakan lengan bayi saat baru lahir (Santai/Kaku)                                        |
| Kaki                             | Pilih pergerakan kaki bayi saat baru lahir (Santai/Kaku)                                          |
| Keadaan Bayi                     | Pilih keadaan bayi saat baru lahir (Tidur,Bangun /Gelisah, Rewel)                                 |
