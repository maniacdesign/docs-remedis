### Cara Pengisian Formulir Pemeriksaan Down Score

| Variable                         | Cara Pengisian                                                                                                                                             |
| -------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Frequensi Nafas                  | Dipilih frekuensi napas bayi saat observasi. (Skor 0 / 1 / 2)                                                                                              |
| Retraksi Dada                    | Dipilih tingkat keparahan tarikan dinding dada ke dalam saat bernapas. (Skor 0 / 1 / 2)                                                                    |
| Sianosis                         | Dipilih kondisi warna kebiruan pada kulit/mukosa. (Skor 0 / 1 / 2)                                                                                         |
| Air Entry                        | Dipilih hasil auskultasi suara napas pada paru-paru. (Skor 0 / 1 / 2)                                                                                      |               
| Merintih                         | Dipilih kondisi suara merintih (grunting) yang menunjukkan kesulitan bernapas. (Skor 0 / 1 / 2)                                                            |
| Down Score (Total Skor)          | Dipilih kondisi suara merintih (grunting) yang menunjukkan kesulitan bernapas. (Skor 0 / 1 / 2)                                                            |
| Kesimpulan                       | Berisi hasil interpretasi tingkat distres pernapasan berdasarkan total skor                                                                                |
