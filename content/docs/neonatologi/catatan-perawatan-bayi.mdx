| Variable                         | Cara Pengisian                                                                                                                                             |
| -------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Tanggal                          | Diisi tanggal saat observasi/pemantauan dilakukan                                                                                                          |
| Waktu Pengisian                  | Diisi jam saat observasi dilakukan, dicentang atau diisi nilai di kolom jam yang sesuai.                                                                   |
| Minum                            | Bagian ini biasanya mencatat volume atau frekuensi minum. Jika kolom Minum berada di bawah Waktu Pengisian, diisi frekuensi atau volume pada jam tersebut. |
| Berat                            | Diisi berat badan bayi saat penimbangan (dalam Gram).                                                                                                      |
| Nafas                            | Diisi nilai frekuensi pernapasan bayi (dalam / Menit).                                                                                                     |
| Suhu                             | Diisi nilai suhu tubuh bayi (dalam â„ƒ).                                                                                                                    |
| Skala                            | Diisi nilai skala nyeri/distres yang diobservasi pada bayi (misal: skor FLACC, NIPS, dll.).                                                                |
| Minum Asi                        | Dipilih Ya atau Tidak untuk menunjukkan apakah bayi menerima ASI.                                                                                          |
| Tambahan/Pengganti               | Dipilih Ya atau Tidak untuk menunjukkan apakah bayi menerima susu formula/cairan tambahan.                                                                 |
| Defekasi                         | Dipilih Ada jika bayi buang air besar (BAB) pada periode observasi                                                                                         |
| Muntah                           | Dipilih Ada jika bayi mengalami muntah pada periode observasi                                                                                              |
| Tidur                            | Dipilih kondisi tidur bayi (Normal / Gelisah)                                                                                                              |
| Tangis                           | Dipilih karakteristik tangisan bayi (Kuat/Lemah/Merintih)                                                                                                  |
| Umbilikus                        | Dipilih kondisi tali pusat: Tidak Bau (normal) atau Bau (indikasi infeksi).                                                                                |
| Lain - lain/Obat                 | Dipilih Ya atau Tidak. Jika Ya, diisi detail catatan penting lain atau obat yang diberikan pada periode tersebut.                                          |
