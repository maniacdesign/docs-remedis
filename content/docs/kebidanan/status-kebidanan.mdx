| Leopoid 2                         | Pilih posisi pungggung janin (punggung kanan/kiri). Contoh : Kanan: teraba bagian keras memanjang (punggung); Kiri: bagian kecil (ekstremitas)                                    |

| VT Oleh                           | Diisi nama petugas yang melakukan pemeriksaan per vaginam (VT)                                                                                                                    |
| Hasil                             | Diisi hasil pemeriksaan per vaginam (VT) (misal: Pembukaan, effacement, Hodge, ketuban, presentasi).                                                                              |
| Bagian Tubuh                      | Pilih salah satu bagian tubuh untuk dilakukan pemeriksaan lebih (Kepadala dan Leher/Dada/Abdomen/Tulang Belakang/Genitalia/Ekstremitas)                                           |
