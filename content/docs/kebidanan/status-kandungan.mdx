| G. Keluhan Lain Saat Haid        | Diisi dengan keluhan lain pasien yang dialami saaat menstruasi. Contoh : Pusing (migrain) dan mual pada hari pertama                                                                                                                 |
| Pendarahan Diluar Haid           | Diisi apabila pasien pernah mengalami pendarahan                                                                                                                                   |
| Jika Pernah, Kapan               | Diisi waktu kejadian apabila pernah mengalami pendarahan diluar haid dan isi banyaknya pendarahan                                                                                  |
| Intensitas Pendarahan            | Pilih intensitas pendarahan (terus menerus/hilang timbul)                                                                                                                          |

| Variable         | Cara Pengisian                                                                                             |
| ---------------- | ---------------------------------------------------------------------------------------------------------- |
| Skala GCS        | Diisikan sesuai dengan tingkat kesadaran pasien berdasarkan respons mata, suara, dan gerak.                |
| Kesadaran        | Pilih kesadaran pasien yang sesuai dengan kondisi pasien Composmentis/Apatis/Samnolen/Stupor/Coma          |
| Keadaan Umum     | Diisi keadaan umum pasien yang sesuai kondisi pasien                                                       |
| Nadi             | Diisi dengan jumlah denyut nadi pasien dalam satu menit (x/menit). Contoh : 80 /menit                      |
| Tensi            | Diisi dengan hasil tekanan darah (tensi) pasien.  Contoh : 120/80 mmhg                                     |
| Suhu             | Diisi dengan tingkat suhu pasien yang telah diperiksa. Contoh : 38'C                                       |
| Respiratory Rate | Diisi dengan jumlah frekuensi napas pasien dalam satu menit. Contoh : 20/menit                             |
| SPO2             | Diisi dengan persentase saturasi oksigen dalam darah. Contohnya : 98%                                      |
| Tinggi Badan     | Diisi dengan tinggi badan pasien                                                                           |
| Berat Badan      | Diisi dengan berat badan pasien                                                                            |
