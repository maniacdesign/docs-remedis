| Variable                                                                                                       | Cara Pengisian                                                                                                                           |
| -------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------        |
| 1. Kebutuhan Alat Bantu Nafas                                                                                                                                                                                                                             |
| Bayi sudah tidak membutuhkan alat bantu nafas ventilator atau ventilasi tekanan positif                        | Dipilih Ya jika  sudah sepenuhnya bernapas spontan tanpa alat bantu napas invasif (ventilator) atau non-invasif (CPAP/NIV)               |
| 2. Kebutuhan Penggunaan Monitor dan Observasi Ketat                                                                                                                                                                                                       |
| A. Bayi sudah tidak membutuhkan monitor dan observasi ketat untuk kegagalan sistem kardiovaskuler (syok)       | Dipilih Ya jika kondisi kardiovaskuler  sudah stabil dan tidak berisiko mengalami syok                                                   |
| B. Bayi sudah tidak membutuhkan terapi vasoaktif seperti dobutamin, dopamine, dan adrenaline                   | Dipilih Ya jika  sudah tidak bergantung pada obat-obatan penunjang tekanan darah/fungsi jantung.                                         |
| C. Bayi sudah tidak menggunakan terapi obat sedasi dalam seperti medazolam                                     | Dipilih Ya jika  tidak lagi memerlukan sedasi tingkat dalam (indikator stabilitas neurologis dan bebas prosedur invasif).                |
| 3. Perawatan dan Pemantauan Pasca Operasi                                                                                                                                                                                                                 |
| Bayi sudah tidak membutuhkan observasi ketat pasca operasi                                                     | Dipilih Ya jika fase kritis pasca operasi sudah terlewati dan kondisi pasien stabil.                                                     |
| Kesimpulan                                                                                                     | Kolom ini akan diisi interpretasi apakah memenuhi semua kriteria Ya untuk step-down.                                                     |


#### 3. Kesimpulan

  src="/keluarnicu3.png"

| Variable                                                                           | Cara Pengisian                                                                                                                                             |
| ---------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------        |
| Berdasarkan Kondisi Diatas, Maka Pasien Memenuhi Indikasi Untuk Keluar Dari:       | Diisi narasi atau kesimpulan akhir yang menyatakan   sudah stabil untuk ditransfer.                                                                        |
| Ruang Perinatologi Level                                                           | Diisi level perawatan perinatologi tempat   dirawat saat ini Contoh : Ruang Perinatologi Level II (Perawatan Intensif Menengah)..                          |
| Alat Transport Yang Digunakan                                                      | Dipilih alat transportasi yang akan digunakan saat memindahkan  (Inkubator jika masih perlu kontrol suhu ketat, Box Bayi jika sudah stabil).               |
| Pendamping Selama Transfer                                                         | Dipilih/dicentang profesi petugas yang akan mendampingi  selama proses transfer/pemindahan.                                                                |
| Dokter                                                                             | Dipilih jika dibutuhkan pendampingan oleh Dokter (risiko tinggi).                                                                                          |
| Perawat / Bidan                                                                    | Dipilih jika dibutuhkan pendampingan oleh Perawat/Bidan.                                                                                                   |
| Transporter                                                                        | Dipilih jika dibutuhkan bantuan tenaga Transporter/Petugas Logistik.                                                                                       |
| Alat Medis                                                                         | Dipilih Ya jika selama transfer  perlu didampingi dengan peralatan medis (misal: tabung oksigen, suction), atau Tidak jika tidak ada kebutuhan khusus.     |
