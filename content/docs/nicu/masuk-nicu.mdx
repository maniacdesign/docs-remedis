| Variable                                                  | Cara Pengisian                                                                                                            |
| --------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------        |
| Denyut Jantung                                            | Pilih sesuai kondisi pasien                                                                                               |
| Nadi Arteri Radialis Teraba Lembut atau Tidak Teraba      | Pilih sesuai kondisi pasien                                                                                               |
| Waktu Pengisian Kapiler > 3 Detik                         | Pilih sesuai kondisi pasien                                                                                               |
| Suhu Axilla < 36.5 - > 37.5 ℃                            | Pilih sesuai kondisi pasien                                                                                               |
| Respirasi < 40 - > 60 x/menit                             | Pilih sesuai kondisi pasien                                                                                               |
| Down Score >= 4                                           | Pilih sesuai kondisi pasien                                                                                               |

#### 4. Pemeriksaan Laboratorium
<img
  src="/masuknicu4.png"
  alt="Contoh tampilan form Kriteria Masuk NICU"
  width="750"
  style={{
    display: "block",
    border: "2px solid blue",
  }}
/>


| Variable                                                  | Cara Pengisian                                                                                                            |
| --------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------        |
| Kadar Haemoglobin < 6 g/dL                                | Pilih sesuai hasil pemeriksaan laboratorium pasien                                                                        |
| Kadar Haematrokit > 65%                                   | Pilih sesuai hasil pemeriksaan laboratorium pasien                                                                        |
| Kadar Leukosit < 50.000 /mL                               | Pilih sesuai hasil pemeriksaan laboratorium pasien                                                                        |
| PH < 7.2 atau > 7.7                                       | Pilih sesuai hasil pemeriksaan laboratorium pasien                                                                        |
| PaO2 < 80 mmHg                                            | Pilih sesuai hasil pemeriksaan laboratorium pasien                                                                        |
| PaCO2 > 50 mmHg                                           | Pilih sesuai hasil pemeriksaan laboratorium pasien                                                                        |
| HCO3- < 15                                                | Pilih sesuai hasil pemeriksaan laboratorium pasien                                                                        |
| Saturasi Oksigen (SpO2) 80%                               | Pilih sesuai hasil pemeriksaan laboratorium pasien                                                                        |
| Kadar Glukosa Serum < 45 mg/dl atau > 150 mg/dl           | Pilih sesuai hasil pemeriksaan laboratorium pasien                                                                        |
| Kadar Natrium Serum < 110 mEq/L atau > 160 mEq/L          | Pilih sesuai hasil pemeriksaan laboratorium pasien                                                                        |
| Kadar Kalium Serum < 2.0 mEq/L atau > 6.0 Eq/L            | Pilih sesuai hasil pemeriksaan laboratorium pasien                                                                        |
| Kadar Kalsium Serum < 6 mg/L atau > 15 mg/L>              | Pilih sesuai hasil pemeriksaan laboratorium pasien                                                                        |
