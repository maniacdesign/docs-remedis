## Catatan Keperawatan NICU 

#### 1. Input
<img
  src="/catatannicu1.png"
  alt="Contoh tampilan form Kriteria Masuk NICU"

  style={{

    border: "2px solid blue",

/>

| Variable                                             | Cara Pengisian                                                                                                                                                      |
| ---------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------        |
| Waktu Pengisian                                      | Diisi dengan tanggal dan jam saat pengkajian/catatan keperawatan ini dibuat.                                                                                        |
| Suhu                                                 | Diisi nilai suhu tubuh saat observasi (dalam Â°C).                                                                                                                   |
| Heart Rate                                           | Diisi nilai frekuensi denyut jantung (dalam kali/menit).                                                                                                            |
| Sp02                                                 | Diisi nilai saturasi oksigen (dalam persentase).                                                                                                                    |
| Infus                                                | Diisi volume atau jenis cairan infus yang masuk selama periode pencatatan.                                                                                          |
| Syringe Pump                                         | Diisi volume atau jenis cairan/obat yang dimasukkan melalui Syringe Pump (pompa spuit).                                                                             |
| Obat Injeksi                                         | Diisi nama dan dosis obat yang diberikan melalui suntikan/injeksi.                                                                                                  |
| Obat Oral                                            | Diisi nama dan dosis obat yang diberikan melalui mulut/oral.                                                                                                        |
| Mode Ventilator                                      | Diisi mode ventilator yang sedang digunakan (jika terpasang ventilator), serta parameter kunci (misal: SIMV, Rate, PEEP)                                            |

| Variable                                             | Cara Pengisian                                                                                                                                                      |
| ---------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------        |
| Urine                                                | Diisi volume urine yang keluar selama periode pencatatan (dalam ml).                                                                                                |
| Draine                                               | Diisi volume cairan yang keluar dari selang drainase                                                                                                                |
| Muntah                                               | Diisi frekuensi dan perkiraan volume/jenis muntahan yang terjadi.                                                                                                   |
| Bab                                                  | Diisi frekuensi dan konsistensi/warna buang air besar (BAB) (misal: 2x, lunak, hijau).                                                                              |
