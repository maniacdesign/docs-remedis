## Catatan Keperawatan NICU 

#### 1. Input
<img
  src="/catatannicu1.png"
  alt="Contoh tampilan form Kriteria Masuk NICU"
  width="750"
  style={{
    display: "block",
    border: "2px solid blue",
  }}
/>

| Variable                                             | Cara Pengisian                                                                                                                                                      |
| ---------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------        |
| Waktu Pengisian                                      | Diisi dengan tanggal dan jam saat pengkajian/catatan keperawatan ini dibuat.                                                                                        |
| Suhu                                                 | Diisi nilai suhu tubuh saat observasi (dalam °C). Contoh : 38°C                                                                                                     |
| Heart Rate                                           | Diisi nilai frekuensi denyut jantung (dalam x/menit). Contoh : 80 /menit                                                                                            |
| Sp02                                                 | Diisi nilai saturasi oksigen (dalam persentase). Contohnya : 98%                                                                                                    |
| Infus                                                | Diisi volume atau jenis cairan infus yang masuk selama periode pencatatan. Contoh : D10% 2.5 ml/jam                                                                 |
| Syringe Pump                                         | Diisi volume atau jenis cairan/obat yang dimasukkan melalui Syringe Pump (pompa spuit). Contoh : Dobutamine 10 mcg/kg/menit                                         |
| Obat Injeksi                                         | Diisi nama dan dosis obat yang diberikan melalui suntikan/injeksi. Contoh : Ceftazidime 100mg (IV)                                                                  |
| Obat Oral                                            | Diisi nama dan dosis obat yang diberikan melalui mulut/oral. Contoh : Nystatin 1 ml (Oral)                                                                          |
| Mode Ventilator                                      | Diisi mode ventilator yang sedang digunakan (jika terpasang ventilator), serta parameter kunci (misal: SIMV, Rate, PEEP)                                            |

| Variable                                             | Cara Pengisian                                                                                                              |
| ---------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------|
| Urine                                                | Diisi volume urine yang keluar selama periode pencatatan (dalam ml). Contoh : 10ml                                          |
| Draine                                               | Diisi volume cairan yang keluar dari selang drainase. Contoh : 5 ml                                                         |
| Muntah                                               | Diisi frekuensi dan perkiraan volume/jenis muntahan yang terjadi. Contoh : 2 ml (susu belum tercerna)                       |
| Bab                                                  | Diisi frekuensi dan konsistensi/warna buang air besar (BAB) (misal: 2x, lunak, hijau).                                      |
