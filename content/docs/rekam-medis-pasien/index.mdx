| Variable                                             | Cara Pengisian                                                                                                               |
| ---------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------  |
| Nama Ibu                                             | Nama lengkap ibu kandung, Contoh : Ny. Sari Rahayu                                                                           |
| Usia                                                 | Usia ibu saat melahirkan, Contoh : 28 Tahun                                                                                  |
| Status: Dirawat                                      | Apakah ibu saat ini juga sedang dirawat di RS? Pilih Iya atau Tidak                                                          |
| Kehamilan                                            | Usia kehamilan (Minggu) dan Status Gravida (G), Para (P), Abortus (A), Contoh : 38 Minggu. G2 P1 A0 (Hamil kedua, anak pertama hidup, tidak pernah keguguran).                                                 |
| Penolong Partus                                      | Tenaga medis yang membantu kelahiran, Contoh : Dokter Obgyn / Bidan                                                          |
| Komplikasi                                           | Masalah medis selama kehamilan, Contoh : Preeklampsia / Diabetes Gestasional / Tidak ada.                                    |
| Tanggal Lahir                                        | Waktu kelahiran bayi (Tanggal & Jam).                                                                                        |
